00 THE POINT OF VIEW

Less mystique.
More contact with reality.

A document is useful once its ideas can be found, connected, and challenged. The workbench turns unstructured text into a transparent internal representation instead of hiding the path behind a single answer.

The loop is: ingest → infer → inspect → improve → project. The graph is not the final truth. It is a living hypothesis with evidence attached and a revision history.

05THE HOUSE RULES
